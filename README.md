# Chat io
Simple chat built with socket.io

### Technologies
* [Node.js](https://nodejs.org/en/)
* [Express](https://expressjs.com/)
* [Ejs](https://ejs.co/)
* [Socket.io](https://socket.io/)

### Install 
In the project folder install the dependencies with
```bash
npm install
```
Start the server
```bash
node index.js
```
